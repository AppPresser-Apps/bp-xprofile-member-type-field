Buddypress xProfile Member Type Field adds member type xProfile field so a user can set a type.

== Description ==


== Installation ==

1. Upload `bp-xprofile-member-type-field` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit Dashboard-> Users-> Profile Fields to add/edit/delete the profile field.
